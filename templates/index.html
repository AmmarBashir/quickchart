Create a Chart Image by URL

This service lets you quickly generate static charts for inclusion in websites, emails, powerpoints, and more. A single URL generates a fully customizable chart.

Creating a chart is easy. Define your chart in an image URL:

<img src="https://quickchart.io/chart?c={type:'bar',data:{labels:['January','February','March','April','May'], datasets:[{label:'Dogs',data:[50,60,70,180,190]},{label:'Cats',data:[100,200,300,400,500]}]}}">

How it Works

The charts are defined by a JSON object. The JSON object follows all the rules of popular open-source graphing library Chart.js. Simplify follow the documentation to define your chart

Once you have the JSON, render it as an image with <img src="https://quickchart.io/chart?c=[[insert json here]]">. Try it below!

Customizing your Chart

You can use any customization options that you're used to from Chart.js. The best place to learn the extent of customization options is through the documentation.

We can get very creative. The example below uses custom background colors, title, stacked bars, and data labels: 

{
  type: 'bar',
  data: {
    labels: ['January', 'February', 'March', 'April', 'May'],
    datasets: [{
      label: 'Dogs',
      backgroundColor: 'chartreuse',
      data: [ 50, 60, 70, 180, 190 ]
    }, {
      label: 'Cats',
      backgroundColor: 'gold',
      data: [ 100, 200, 300, 400, 500 ]
    }]
  },
  options: {
    title: {
      display: true,
      text: 'Total Revenue (millions)',
      fontColor: 'hotpink',
      fontSize: 32,
    },
    scales: {
      xAxes: [{stacked: true}],
      yAxes: [{stacked: true}],
    },
    plugins: {
      datalabels: {
        display: true,
        font: {
          style: 'bold',
        },
      },
    },
  },
}

A couple things to remember when customizing your chart:

  • Interactivity and animations are not supported (this service produces static images)
  • We recommend that you URL-encode the entire JSON block. If not URL-encoded, you must escape all hex color codes (e.g. #ff0000 → %23ff0000, or just use the rgb(255, 0, 0) equivalent).
