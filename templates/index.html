Create a Chart Image with a URL

One url generates a chart. This service lets you quickly generate static charts for inclusion in websites, emails, powerpoints, and more.

Creating a chart is easy. Define your chart in an image URL:

<img src="https://quickchart.io/chart?c={type:'bar',data:{labels:['January','February','March','April','May'], datasets:[{label:'Dogs',data:[50,60,70,180,190]},{label:'Cats',data:[100,200,300,400,500]}]}}">

How it Works

The charts are defined by a JSON object. The JSON object follows all the rules of popular open-source graphing library Chart.js. Simplify follow the documentation to define your chart: 

{
  type: 'bar',
  data: {
    labels: ['January', 'February', 'March', 'April', 'May'],
    datasets: [{
      label: 'Dogs',
      data: [ 50, 60, 70, 180, 190 ]
    }, {
      label: 'Cats',
      data: [ 100, 200, 300, 400, 500 ]
    }]
  }
}

Once you have the JSON, render it as an image with <img src="https://quickchart.io/chart?c=[[insert json here]]">. Here's what we get:

Your chart is very customizable. By changing type: bar to type: line, for example, we can instantly produce the equivalent line graph:

<img src="https://quickchart.io/chart?c={type:'line',data:{labels:['January','February','March','April','May'], datasets:[{label:'Dogs',data:[50,60,70,180,190]},{label:'Cats',data:[100,200,300,400,500]}]}}">
